## 🛠 About one-click-bios

one-click-bios lets you restart your Windows computer and enter the BIOS or UEFI settings with just one click. This tool adds a shortcut to your Start Menu. You do not need to press any keys while your computer boots. This can save time and help if your keyboard or function keys do not work during startup.

The program runs on Windows and uses built-in Windows features to trigger the restart. It automates a process that usually requires navigating menus or pressing special keys during boot.

## 🧰 How It Works

one-click-bios uses Windows commands to tell your computer to restart into firmware settings. Windows 10 and newer include a command called `shutdown /r /fw /t 0`. This tells the system to reboot and open the BIOS or UEFI setup screen.

The shortcut runs this command with administrator rights so it can access the required functions. all you have to do is click the shortcut.

## ✋ Manual BIOS Access (Optional)

If you prefer not to use one-click-bios or the tool does not work on your PC, you can access BIOS manually:

1. Restart your PC.  
2. During boot, press the BIOS key (common ones are F2, Delete, or Esc).  
3. If you miss the timing, your PC will boot normally. You must try again.  

One-click-bios removes this guesswork by automating the restart with BIOS entry.
